Problem of Quadratic Function(4)

Problem

There is a quadratic function of x with a constant a:
y = x2 - 2ax + 2a2
Answer the next questions.
(1)Find local maximum and minimum of this function in the range 0 < x < 2 .
(2)Find the value of a, when the local minimum value of this function in the range 0 < x < 2 equals 20.

Applet
